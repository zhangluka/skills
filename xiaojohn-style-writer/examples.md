# 小约翰可汗风格示例

供写稿时参考的句式与段落，严格模仿语气与节奏。

## 让步转折句式（可替换填空）

- 态度不说是守望相助吧，至少也可以说是不共戴天。
- 不能说是家财万贯，至少也可以说是资不抵债。
- 虽然称不上是品学兼优吧，但也可以说是狂嫖滥赌。
- 不能说是德高望重吧，也至少是人厌狗嫌。
- 关系不说是情同手足吧，至少也可以说是不共戴天。

## 人物/国家出场（超长定语 + 称号）

- 非洲民族解放运动的元老、统一劳动党的终身主席、赤道几内亚的终身总统、焚书坑儒的行动派、人类清除计划的实践者、位列非洲三大暴君的五个人之一的赤道几内亚唯一奇迹——马西埃。
- 可仿造结构：头衔1、头衔2、头衔3、戏谑定语、数字梗/排行梗——人名/国名。

## 大厂黑话魔改 + 叙事

美听宗和季辛格政府重视结果导向，对智利的市场环境进行目标对齐，全面布局，发力痛点，打出一套组合拳，最终成功形成了灾难性的生态化反。

可替换：以 XX 为抓手、打通闭环、对齐目标、深耕痛点、打出组合拳、形成生态化反。

## 通辽宇宙 + 黑色幽默

- 面积相当于 2.75 个通辽，但人口只有区区 60 万，跟回龙观差不多。
- 穷得荡气回肠，物理意义上的敲骨吸髓。
- 空气比薯片还多（形容地广人稀或某种荒诞对比）。

## 结尾升华模板（理想主义 + 唏嘘）

- 人的肉体必然会湮灭，但精神会永远存在。
- 屠龙者变成了龙，自然是要死掉的，但我们不能忘记他曾经为了屠龙所做过的努力。他的躯体已经死掉，但在他年轻时向恶龙挥舞长剑的那一刻，他的光荣是永恒的。
- 可延伸：劳动者/弱者/理想主义者的视角，历史镜像，哀而不伤，避免说教。

## 废话文学 / 倒装 / 梗

- ……属于是。
- 赢麻了 / 都赢麻了。
- 谐音梗、歇后语、网络热梗的改编（根据主题自然植入）。

写作时从以上元素中选取搭配，保持北方口语感和信息密度，结尾务必升华。
