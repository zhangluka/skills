# grain-plan-writer

谷雨计划公众号长文写作 skill。

## 一句话

装上之后，Agent 用谷雨计划的口吻和节奏写公众号长文——实测记录、场景拆解、热点解读、项目进展。

## 适合

- 你有一个AI相关的公众号，想保持"动手验证、不空谈"的调性
- 你有素材（PDF、语音转文字、新闻链接、实验记录），想让它帮你写成文章
- 你想要一个有自检体系的写作流程，避免AI味

## 不适合

- 你想要的是"爆款标题公式""10万+模板"——这个skill不追流量，追真实
- 你想要的是纯文学/散文风格——谷雨的风格偏实操、偏记录

## 特色

- **五种文章类型**：实测报告、场景拆解、热点解读、项目记录、工具对比
- **四层自检体系**：L1 硬性规则 → L2 结构风格 → L3 内容质量 → L4 整体感觉
- **绝对禁区清单**：禁用词、禁用句式、禁用开头，精确到每个容易暴露AI味的表达
- **AI协作边界**：明确哪些事该交给AI、哪些必须自己来
